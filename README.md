# ejecutar en desarrollo

1. clonar el repositorio
2. ejecutar 
   ```
   yarn install
   ```
3. tener Nest CLI instalado 
    ``` 
    npm i -g @nestjs/cli
    ```
4. Levantar la base de datos en 
   
    ```
    docker-compose up -d 
    ```

    ## Stack
  * MongoDB
  * Nest
